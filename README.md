# Online-Retail-Sales-Analysis-Excel-DataAnalyst_Project-
Vrinda Store Sales Analysis Dashboard

This project presents a Sales Analysis Dashboard for Vrinda Store created using Microsoft Excel. The dashboard analyzes sales data to uncover insights about customer behavior, order trends, sales channels, and product performance. It helps in understanding key business metrics and identifying factors that drive sales growth.

The dashboard includes important KPIs such as Total Sales, Total Orders, Average Order Value, and Top Sales Channel. It also provides visual insights into sales trends over time, gender-based sales distribution, order status breakdown, top-performing states, and sales contribution by different channels.

Interactive filters such as Month, Channel, and Category allow users to dynamically explore the data and analyze performance across different dimensions. This project demonstrates how Excel can be used for data analysis, visualization, and business intelligence reporting.

Key Insights

Women customers contribute a larger share of total sales.

Most orders are successfully delivered with very few cancellations.

Certain states generate significantly higher revenue compared to others.

Online marketplaces like Amazon and Flipkart drive a large portion of sales.
